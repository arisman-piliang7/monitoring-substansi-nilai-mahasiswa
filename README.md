# 📊 Monitoring Substansi Nilai Mahasiswa

Aplikasi web yang menjelaskan substansi nilai mahasiswa secara detail agar tidak ada keraguan dalam diri masing-masing 😄

## Fitur

- **Tabel Konversi Nilai** — Daftar lengkap nilai huruf (A–E), rentang angka, bobot poin, dan kualitas capaian.
- **Konverter Nilai Cepat** — Masukkan nilai angka (0–100) dan langsung dapatkan nilai huruf beserta bobotnya.
- **Substansi & Makna Setiap Nilai** — Penjelasan detail tentang arti dan ekspektasi untuk setiap nilai huruf.
- **Predikat Kelulusan** — Informasi predikat Cum Laude, Sangat Memuaskan, Memuaskan, dan Cukup beserta syaratnya.
- **Kalkulator IPK** — Hitung IPK secara otomatis dari daftar mata kuliah, nilai huruf, dan SKS.
- **Komponen Penilaian** — Penjelasan bobot umum tugas, UTS, UAS, dan kehadiran.

## Cara Menjalankan

Tidak diperlukan instalasi atau server khusus. Cukup buka file `index.html` di browser:

```bash
# Clone repository
git clone https://github.com/arisman-piliang7/monitoring-substansi-nilai-mahasiswa.git
cd monitoring-substansi-nilai-mahasiswa

# Buka di browser (macOS/Linux)
open index.html

# atau (Windows)
start index.html
```

Atau langsung buka melalui GitHub Pages jika telah diaktifkan.
